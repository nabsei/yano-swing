YANO SWING — BETA (macOS only)
========================================

This is an early beta build. It is not yet notarized by Apple, so macOS
will show a security warning the first time you use it. This is normal
for an early-access indie plugin — here's how to get past it in ~30 seconds.

Universal binary: works on both Apple Silicon (M1/M2/M3...) and Intel Macs.

NOTE: Yano Swing is a MIDI effect (it retimes note events), not an audio
effect. Insert it on a MIDI or instrument track that supports MIDI FX,
ahead of your instrument -- not on an audio track.

INSTALL
-------

1. Copy "Yano Swing.vst3" to:
   ~/Library/Audio/Plug-Ins/VST3/

2. Copy "Yano Swing.component" to:
   ~/Library/Audio/Plug-Ins/Components/

   (You can also use "Yano Swing.app" on its own — no DAW needed —
   if you just want to try it with a MIDI controller.)

3. Open Terminal (Applications > Utilities > Terminal) and run:

   xattr -cr ~/Library/Audio/Plug-Ins/VST3/"Yano Swing.vst3"
   xattr -cr ~/Library/Audio/Plug-Ins/Components/"Yano Swing.component"
   xattr -cr /path/to/"Yano Swing.app"

   This removes the "downloaded from the internet" flag that would
   otherwise make Gatekeeper block an unsigned plugin. It does not
   disable your Mac's security — it just tells macOS this specific
   file is one you trust.

4. Rescan plugins in your DAW (or restart it) — Yano Swing should
   now show up under your MIDI effects.

TROUBLESHOOTING
----------------

If your DAW still refuses to load it: open System Settings > Privacy &
Security, scroll down — you should see a message about "Yano Swing"
being blocked, with an "Open Anyway" button. Click it once.

If your DAW doesn't show it in the plugin list at all, check that you're
looking in the MIDI effects category, not audio effects — some DAWs
list them separately.

QUESTIONS / BUGS
-----------------

This is a beta. If something sounds wrong or crashes your DAW, please
report it — beta feedback directly shapes what ships in v1.

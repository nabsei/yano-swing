YANO SWING — BETA (Windows)
=====================================

This is an early beta build, not yet code-signed. Windows SmartScreen may
show a warning the first time you run it — this is normal for an early
indie plugin build, here's how to get past it.

NOTE: Yano Swing is a MIDI effect (it retimes note events), not an audio
effect. Insert it on a MIDI or instrument track that supports MIDI FX,
ahead of your instrument -- not on an audio track.

INSTALL
-------

1. Copy the "Yano Swing.vst3" folder to:
   C:\Program Files\Common Files\VST3\

2. Rescan plugins in your DAW (or restart it) — Yano Swing should
   now show up under your MIDI effects.

   (You can also run "Yano Swing.exe" directly — no DAW needed —
   if you just want to try it with a MIDI controller.)

IF WINDOWS SMARTSCREEN BLOCKS IT
---------------------------------

When you first run the .exe (or your DAW loads the plugin), you may see
"Windows protected your PC". Click "More info", then "Run anyway".
This appears because the build isn't yet signed with a paid code-signing
certificate — it does not mean the file is unsafe, just that Windows
doesn't yet recognize the publisher.

QUESTIONS / BUGS
-----------------

This is a beta. If something sounds wrong or crashes your DAW, please
report it — beta feedback directly shapes what ships in v1.
